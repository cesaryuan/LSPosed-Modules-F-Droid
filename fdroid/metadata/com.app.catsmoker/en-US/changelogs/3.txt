**Release Notes for CatSmoker App - Version [1.3]**

### What's New:

Changes:
- Enabled opening non-GitHub pages outside the app.
- Added a "No Internet" page to display when offline.
- Introduced support for the Warzone game.
- Enhanced dark mode for a better user experience.
- Removed unnecessary deployments to streamline app updates.
- Implemented analytics to monitor app performance.
- Added back button functionality for easier navigation.
- Improved network connectivity checks to ensure accurate online/offline detection.

Bug Fixes:
- Various bug fixes to enhance app stability and performance.

_Thank you for using CatSmoker!_
